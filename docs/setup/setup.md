## Prerequisites

- Intel i7 gen 9th and above
- 32 GB RAM memory minimum
- NVIDIA RTX 2060
- Ubuntu 18.04
- NVIDIA Driver metapackage from nvidia——driver-460(proprietary,tested) 
- Carla 0.9.10
- Docker-CE version 19.03 and above
- NVIDIA Container Toolkit
